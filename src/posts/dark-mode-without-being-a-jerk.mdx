a few notes on what i learned wiring up dark mode for this portfolio. mostly small things i'd want to remember if i did it again.

## the toggle should feel like a thing

the cheap version is a checkbox. the warm version is a sun-to-moon button with spring physics. it costs maybe twenty extra lines and changes how people feel about the page.

## defaults are a values question

i don't auto-pick based on system preference at first load. i pick light, warm cream, intentional. people who want dark can tap once.
